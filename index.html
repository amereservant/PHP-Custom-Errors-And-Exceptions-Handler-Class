PHP-Custom-Errors-And-Exceptions-Handler-Class
by amereservant & Liam Best

This is a fully customizable PHP 5 error/exception handler class that gives the user far more control in how errors and exceptions are displayed and logged compared to what PHP's defaults offer. It allows different settings to be specified for displaying and logging, user-defined viewer friendly error messages on production sites, and more.
There are two versions, a simple one written solely by Liam Best, which doesn't use any additional files and very minimal configuration options, then another one that is a derivative of Liam's version with more control over it's behavior and configuration options.

This project is currently still being developed, so you can either fork it from here or wait for a release update to be committed. Thanks.
